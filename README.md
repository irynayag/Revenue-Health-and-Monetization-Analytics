# 📊 Revenue Health & Monetization Analytics

## 📝 Опис проекту
Цей проект присвячений комплексному аналізу життєвого циклу користувачів: від етапу проходження онбордингу до моменту здійснення перших платежів.

На основі даних про поведінку користувачів під час онбордингу було розраховано ключові фінансові метрики. Дашборд дозволяє аналізувати зв'язок між демографічними ознаками користувачів, їхньою активністю та грошовими надходженнями. Інструмент надає продуктовим менеджерам можливість проводити факторний аналіз змін доходу та відстежувати рівень відтоку (Revenue Churn) у динаміці.

## 🛠 Технологічний стек
* **SQL:** обробка даних, використання CTEs та Window Functions для аналізу грошових потоків.
* **Tableau:** візуалізація метрик та побудова інтерактивного дашборду.
* **Аналітичні показники:** MRR, ARPPU, LTV, Churn Rate, Expansion MRR, etc.
